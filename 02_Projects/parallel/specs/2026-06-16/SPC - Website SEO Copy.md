# SPC - Website SEO Copy: parallel-projects.net

**Project:** Parallel
**Platform:** Squarespace
**Primary audience:** Wedding planners and event coordinators in Los Cabos
**Language:** English
**Tone:** Luxury/refined + creative/artistic
**Geographic target:** Los Cabos, Baja California Sur

---

## 1. Goals

1. Make parallel-projects.net discoverable for high-value search terms (projection mapping, laser shows, lighting design in Los Cabos) where there is currently zero competition
2. Position Parallel as a bespoke specialty studio — not a production rental house — in the language planners use when sourcing vendors
3. Give planners everything they need to understand, trust, and contact Parallel within a single visit
4. Establish the venue credit list (Acre, Palmilla, Viceroy, Chileno Bay, Amara, Hotel San Cristobal, Paradero, Kimpton Mas Olas) as a trust anchor

**Success criteria:**
- Ranking in top 5 for "projection mapping Los Cabos" within 90 days of implementation
- Ranking in top 5 for "lighting design Cabo San Lucas" within 90 days
- Measurable increase in planner inquiries via contact form

---

## 2. Positioning Statement

> Parallel is Los Cabos' specialist in bespoke lighting design, projection mapping, and laser production. We work with the region's leading planners and venues — not as a production house, but as a creative and technical studio that designs each event from scratch, using whatever gear the vision requires.

---

## 3. Tagline Options

Three directions. Pick one, or combine elements.

**Option A — Location-anchored, functional (strongest for SEO)**
> "Los Cabos' Specialist in Light, Lasers & Projection"

This reads as a credentialed expert claim. It's specific, geo-targeted, and contains the three most searchable service terms. Best for homepage H1.

**Option B — Artistic, minimal**
> "Designed for the moment. Built for Los Cabos."

Luxury-brand feel. Works as a hero subheading or brand line under the logo. Less SEO-direct, but creates stronger emotional resonance.

**Option C — Vision-first**
> "Every event is a different light."

Lyrical. Works well in print, social, and as a secondary tagline. Not SEO-optimized but captures the bespoke, one-of-a-kind nature of the work.

**Recommended approach:** Use Option A as the H1 (for search ranking) and Option B as the subheading directly beneath it (for emotional impact). Option C for social and print.

---

## 4. Voice & Tone Guidelines

**Write as:** A refined specialist who lets the work speak — confident without being boastful, specific without being technical for its own sake.

**Do:**
- Name specific gear (KVant, Unity) when it signals quality
- Name venues when establishing credibility
- Use "we design" / "we produce" — not "we provide" or "we offer"
- Speak to the planner's goal, not Parallel's inventory ("your vision" not "our equipment")
- Keep sentences short and declarative in hero/heading copy; allow slightly more texture in body copy

**Do not:**
- Use production house language ("packages," "rental rates," "quote request")
- List gear specs as a primary sell
- Use clichés: "unforgettable," "magical," "dream wedding"
- Use filler adjectives: "top-quality," "professional," "world-class"

## 7. LocalBusiness Schema (for code injection in Squarespace)

Add this JSON-LD to the Squarespace header code injection field (Settings > Advanced > Code Injection):

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "Parallel",
  "url": "https://parallel-projects.net",
  "email": "parallelprojectsmx@gmail.com",
  "description": "Bespoke lighting design, projection mapping, and laser production studio in Los Cabos, Mexico.",
  "areaServed": {
    "@type": "Place",
    "name": "Los Cabos, Baja California Sur, Mexico"
  },
  "serviceType": [
    "Projection Mapping",
    "Laser Shows",
    "Lighting Design",
    "Custom LED Installations",
    "Decorative Lighting",
    "Event Visuals"
  ],
  "sameAs": [
    "https://www.instagram.com/parallel_mx/"
  ]
}
</script>
```

*Note: Add a physical address and phone number to the schema once confirmed — Google weights those fields for local ranking.*
